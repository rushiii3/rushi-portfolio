OddNotEven Store is a Shopify-based e-commerce platform developed for a client under the agency **References**. While not fully customized, the project includes tailored sections to improve the shopping experience, navigation, and overall design.

## Key Features

- **Custom Sections on Shopify Theme:** Developed and integrated specific sections using Liquid, HTML, and CSS to enhance the default Shopify theme.
- **SEO & Performance Optimizations:** Ensured high-speed performance and improved search engine visibility for increased conversions.
- **Mobile-First Design:** Optimized for a fully responsive shopping experience across all devices.
- **Custom Footer & Mobile Navbar:** Designed a tailored footer and mobile navigation bar for enhanced usability.
